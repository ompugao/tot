# Tot

Manage your todo on your terminal, inspired by [console-task-checker](https://github.com/alice1017/console-task-checker).

## Installation

Add this line to your application's Gemfile:

    gem 'tot'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tot

## Usage

Please see

    $ tot help
